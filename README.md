# 🎓 Prédiction d'Abandon Étudiant

## 📋 Description du Projet
Système de machine learning pour prédire le risque d'abandon scolaire des étudiants, permettant une intervention précoce.

## 🎯 Objectifs
- Identifier les étudiants à risque d'abandon
- Fournir des insights sur les facteurs de risque
- Proposer des interventions personnalisées
- Interface web pour utilisation par le personnel éducatif

## 📊 Dataset
- **Source** : UCI Student Performance Dataset
- **Échantillons** : ~650 étudiants
- **Features** : 33 variables (démographiques, sociales, académiques)
- **Target** : `dropout_risk` (0: faible risque, 1: haut risque) - créée à partir des critères académiques
